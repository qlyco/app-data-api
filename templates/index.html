API Status

If you can view this page, the API is running properly.

Endpoints

GET:

  • / : View this page.
  • /backup : Download the database backup.

      Headers:

    • passkey : A valid SHA256 encrypted access passkey.
  • /seed : Get the default seed for random number generator.
  • /seed/<seed-type> : Get the seed for random number generator (<seed-types>: hourly, daily, weekly, monthly, yearly).
  • /time : Get the current server time.
  • /apps : Get the list of app details.
  • /apps/<app-name> : Get the details of a specific app.
  • /tracker/<app-name>/<year>/<month> : Get the traffic stats of a specific app.
  • /scores/<app-name> : Get the lifetime scores of a specific app.

      Params:

    • limit : Amount of records to be fetched (Default = 10).
  • /scores/<app-name>/<score-type> : Get the leaderboard of a specific app. (<score-types>: daily, weekly, monthly, yearly).

      Params:

    • limit : Amount of records to be fetched (Default = 10).

POST:

  • /tracker/<app-name> : Send the traffic data for a specific app.
  • /scores/<app-name> : Send a new score submission of a specific app.

      JSON:

    • user : Unique username of the user.
    • score : Score to be submitted.
